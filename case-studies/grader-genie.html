GraderGenie — Automated Homework Grading System

A real-time automated homework grading platform with an analytics dashboard, designed to reduce instructor grading time and evaluated comprehensively against Nielsen's 10 usability heuristics.

2024 Solo Developer ~5 weeks JavaScript · Analytics · HCI
Demo Source

The Problem

Manual homework grading is time-consuming and introduces inconsistency. Instructors in large courses spend 40–60% of their time on repetitive grading tasks. An automated system with rule-based and pattern-matching evaluation can handle routine assessments while surfacing analytics to help instructors focus on learning gaps.

Process Pipeline

Assignment Upload
Rubric Configuration
Automated Grading
Result Review
Analytics Dashboard
Export

Key Features

1

Rubric Builder

Instructors define marking criteria with point values and accepted answer patterns (exact match, keyword presence, regex).

2

Automated Grading Engine

Student submissions evaluated against rubric in real time. Confidence scores flag ambiguous answers for human review.

3

Analytics Dashboard

Class-level analytics: score distribution, per-question accuracy heatmap, top misconceptions identified by wrong-answer clustering.

4

Nielsen Heuristics Evaluation

Formal evaluation of all 10 heuristics. Issues documented with severity ratings (0–4). Redesign iterations addressed all severity 3–4 findings.

5

Export & Feedback

Per-student feedback reports generated automatically; batch CSV export for gradebook integration.

Key Learnings

  • Heuristic evaluation before user testing is cost-effective — the formal evaluation caught 8 significant issues that would have required participant recruitment to discover otherwise.
  • Confidence scores on automated grades are essential for instructor trust — a binary pass/fail with no explanation is rejected by instructors.
  • Score distribution charts are the most-used dashboard feature; surface them first.
