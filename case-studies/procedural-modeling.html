Procedural Modeling — L-Systems & Boids Simulation

A real-time 3D environment featuring procedurally generated plant life using L-system grammar rules and an emergent Boids flocking simulation, with live parameter sliders built in OpenGL and C++.

Problem

Procedural content generation is fundamental to games, film, and architectural visualisation. Two of the most elegant procedural techniques — L-systems for organic plant growth and Reynolds' Boids for emergent flocking behaviour — can be implemented from first principles in OpenGL to demonstrate how complex-looking outputs emerge from simple rules.

Key Features

1

L-System Engine

Parametric L-system with configurable axiom and production rules. Supports branching (push/pop turtle state), angles, and iteration depth. Generates diverse plant morphologies (trees, ferns, coral) by varying only the rule strings.

2

Turtle Interpreter

L-system output strings interpreted as 3D turtle commands: move forward, rotate yaw/pitch/roll, push/pop transform stack. Geometry extruded along the turtle path as tapered cylinders.

3

Boids Flocking Simulation

N-body simulation of up to 200 agents following Reynolds' three rules: separation (avoid crowding), alignment (match neighbour velocity), cohesion (steer toward group centre). Each rule has a configurable weight slider.

4

Real-Time Parameter Sliders

Iteration depth, branch angle, boid count, separation/alignment/cohesion weights all adjustable at runtime via FLTK sliders. Changes immediately re-generate geometry or update simulation parameters.

5

Scene Integration

L-system plants and Boids flock coexist in the same 3D scene with perspective camera and orbit controls.

Key Learnings

L-system depth is exponential — depth 6 can produce millions of segments; geometry instancing is essential beyond depth 4.

Boids separation is the most sensitive parameter — too low causes agents to pile up, too high causes fragmentation.

Spatial partitioning (grid hashing) reduces Boids neighbour search from O(n²) to O(n) for large flocks.
